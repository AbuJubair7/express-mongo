# express-mongo

Express and MongoDB CRUD operation practice project built with TypeScript.

## Scripts

- `npm run dev` - run the app in development mode
- `npm run build` - compile TypeScript to `dist`
- `npm start` - run the compiled app

## Environment

- `PORT` - server port, defaults to `3000`
- `MONGODB_URI` - optional MongoDB connection stringThis is an Express and MongoDB CRUD operation practice project.
